DEVELOPER IMAGES
================

To add developer photos to the Developers page, place image files here with these names:

- dev1.jpg (or .png) - Developer 1 photo
- dev2.jpg (or .png) - Developer 2 photo  
- dev3.jpg (or .png) - Developer 3 photo
- dev4.jpg (or .png) - Developer 4 photo

If images are not provided, placeholder images will be shown automatically.

Recommended image specifications:
- Format: JPG or PNG
- Size: 150x150 pixels (square)
- File size: Under 500KB each
